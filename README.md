# Day-29-Introduction-to-Docker
1)What is Docker?
Docker is an open-source platform used to build, ship, and run applications in containers.
It helps developers run applications the same way on any system.
2)Containers vs Virtual Machines?
Containers share the host OS and are lightweight.
Virtual Machines have their own full OS and are heavy.
3)Docker Architecture?
Docker Client – Runs commands 
Docker Daemon – Run in background 
Docker Image – Blueprint of application
Docker Container – Running instance of image
Docker Registry – Stores images
